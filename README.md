A gamified nature app for students to track recycling, earn badges, and complete eco-challenges.

## Features

- **Home Feed**: Track class progress and see what others are recycling.
- **AI Scanner**:
  - **Recycle Mode**: Identifies trash and tells you how to recycle it.
  - **Craft Mode**: Suggests DIY projects from trash items.
  - **Accessibility Mode**: Audio feedback for visually impaired users.
- **Leaderboard**: Compete with your class and school.
- **Profile**: View stats (CO2 saved, trees saved) and collected badges.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the app:
   ```bash
   npx expo start
   ```

## Tech Stack

- React Native (Expo)
- React Navigation (Bottom Tabs)
- Lucide React Native (Icons)
- Expo Speech (Accessibility)
